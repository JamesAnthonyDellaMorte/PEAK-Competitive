# Changelog

All notable changes to PEAK Competitive will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release
- Timer system for tracking climb times
- Leaderboard functionality
- Race mode for competitive multiplayer
- Match statistics tracking
- In-game configuration menu (F3)
- Configurable settings via BepInEx config file

### Features
- Personal best tracking
- Synchronized race timers
- End-screen statistics display
- Customizable UI positioning
- Host-only mod (clients don't need it)

- None yet!
